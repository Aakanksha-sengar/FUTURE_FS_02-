# 🚀 Mini CRM (Full Stack Web App)

A Mini CRM (Customer Relationship Management) web application built using **HTML, CSS, JavaScript, Node.js, Express, and MongoDB Atlas**.

---

## ✨ Features

- ➕ Add new leads (Name, Email, Phone, Message)
- 📋 View all leads
- 💾 Data stored in MongoDB Atlas
- 🌙 Dark Mode / Light Mode toggle
- 🎨 Clean and responsive UI (White + Blue theme)

---

## 🛠️ Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB Atlas

## ⚙️ Installation & Setup

1. Clone the repository
git clone https://github.com/Aakanksha-sengar/FUTURE_FS_02.git

2. Navigate to project folder
   cd FUTURE_FS_02
   
3. Install dependencies
   npm install
   
4. Add MongoDB URI in `server.js`
   "mongodb+srv://aakanksha:Aakanksha%402007@portfolio.t6ot00s.mongodb.net/?retryWrites=true&w=majority"

6. Run the project
   npm start
   
7. Open in browser
   http://localhost:3000/
